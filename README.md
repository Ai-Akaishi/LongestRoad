# LongestRoad

Minecraftのコマンドで、カタンの最長交易路(Longest Road)を見つけだします。  
このワールドの方法では、functionなしで見つけ出せます。(コマブロ設置済み)  
必要コマンド実行数 = 18 + (n + 22) * (n - 1) + 4 (n:街道初期所持数)

# 操作

- スコアボードなどの作成  
  function calc/init
- 最長交易路の計算  
  function calc/main
- 最長交易路の表示  
  function calc/show
- 配置を初期化  
  function demo/start
- １ターン進める  
  function demo/resume
- 10ターン進める  
  function demo/resume10
- 計算して表示する  
  function demo/show
